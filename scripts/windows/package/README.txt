willGo Windows Installer Package
================================

How to install
--------------
1. Extract this package to any folder.
2. Double-click Install-willGo.bat.
3. The installer copies the app to your user profile and creates shortcuts.

What gets installed
-------------------
- App files: %LOCALAPPDATA%\Programs\willGo
- Desktop shortcut: willGo
- Start menu folder: willGo
- Uninstall entry: Apps & Features / Installed apps

Notes
-----
- This package is self-contained for Windows x64 and does not require a separate .NET install.
- User settings and logs stay in %LOCALAPPDATA%\FloatingDeskAssistant.
- To remove the app later, use the Start menu uninstall shortcut or the Installed apps entry.
